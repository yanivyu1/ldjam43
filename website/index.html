Hire Power

Play | Play extended* version | Play full* version
Screenshots | Source Code | Credits | Self-Promotion

This game is our submission to Ludum Dare 43 Jam, theme: Sacrifices must be made.

* The extended version is longer, with more difficult levels.
* The full version contains levels that can't be solved because we didn't have time to finish them :)

You may skip levels using Shift+N, restart the level using Shift+R, or go back to the previous level using Shift+P.

Good prophets are so hard to find. I, Wana-Haya'yu, the magnificent corporate volcano god, want to hire YOU!
Yes, you, the prophet for hire, will be tested.
How many followers can you gather? How many will you joyously integrate into my Sacred Lava™?
Do you think you have what it takes to get the job? Apply now to spread the eternal redemption of the executive flames.
